# 변경 이력 (Changelog)

HNSMES 시스템의 버전별 변경 이력입니다.

## 버전 규칙

버전 번호는 `MAJOR.MINOR.PATCH` 형식을 따릅니다.

| 위치 | 설명 | 예시 |
|------|------|------|
| MAJOR | 주요 기능 변경, 호환성 없는 변경 | 1.0.0 → 2.0.0 |
| MINOR | 기능 추가, 하위 호환성 유지 | 1.0.0 → 1.1.0 |
| PATCH | 버그 수정, 하위 호환성 유지 | 1.0.0 → 1.0.1 |

---

## [2.1.0] - 2024-12-15

### 추가

- 생산계획 시뮬레이션 기능 (PLAN015)
- AI 기반 불량예측 대시보드 (MST026)
- 모바일 생산실적 입력 앱 연동
- 다중 언어 지원 (영어, 중국어)

### 개선

- 대시보드 로딩 속도 40% 향상
- 리포트 출력 속도 개선
- 재고 조회 성능 최적화

### 수정

- 설비비가동 등록 시 간헐적 오류 수정
- 수입검사 LOT 중복 체크 로직 수정
- 작업지시 확정 시 권한 체크 오류 수정

### 보안

- 비밀번호 정책 강화 (12자리 이상)
- 2차 인증(OTP) 지원 추가
- 세션 관리 로직 개선

## 업그레이드 가이드

### 1.x → 2.0 업그레이드

```
1. 데이터베이스 백업
2. 기존 프로그램 제거
3. 신규 버전 설치
4. 데이터 마이그레이션 실행
5. 설정 파일 복원
6. 기능 테스트
```

### 주의사항

- 업그레이드 전 반드시 전체 백업 수행
- 업그레이드는 계획된 점검 시간에 진행
- 롤백 계획 사전 수립
