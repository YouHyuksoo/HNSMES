# 용어집 (Glossary)

## 개요

본 문서는 HAENGSUNG HNSMES 시스템에서 사용되는 주요 용어와 약어를 설명합니다.

---

## 공통 약어 (Common Abbreviations)

| 약어 | 영문 | 한글 | 설명 |
|------|------|------|------|
| **MES** | Manufacturing Execution System | 제조실행시스템 | 생산 현장의 작업을 관리하는 시스템 |
| **ERP** | Enterprise Resource Planning | 전사적자원관리 | 기업 전체 자원을 통합 관리하는 시스템 |
| **WMS** | Warehouse Management System | 창고관리시스템 | 창고 운영을 관리하는 시스템 |
| **QMS** | Quality Management System | 품질관리시스템 | 품질 관련 프로세스를 관리하는 시스템 |
| **SCM** | Supply Chain Management | 공급망관리 | 공급망 전반을 관리하는 시스템 |
| **APS** | Advanced Planning and Scheduling | 고급계획및일정관리 | 생산 계획 및 일정 최적화 시스템 |
| **WIP** | Work In Process | 재공품 | 생산 중인 제품 또는 반제품 |
| **BOM** | Bill of Materials | 자재명세서 | 제품을 구성하는 부품 목록 |
| **SOP** | Standard Operating Procedure | 표준작업절차 | 표준화된 작업 방법 |

## 기술 용어 (Technical Terms)

### 개발 기술

| 용어 | 설명 |
|------|------|
| **WinForms** | Windows Forms, Windows 데스크톱 애플리케이션 개발 프레임워크 |
| **WCF** | Windows Communication Foundation, 분산 애플리케이션 통신 프레임워크 |
| **WPF** | Windows Presentation Foundation, Windows UI 프레임워크 |
| **ORM** | Object-Relational Mapping, 객체와 데이터베이스 간 매핑 기술 |
| **SOAP** | Simple Object Access Protocol, XML 기반 통신 프로토콜 |
| **REST** | Representational State Transfer, HTTP 기반 API 아키텍처 스타일 |
| **JSON** | JavaScript Object Notation, 경량 데이터 교환 형식 |
| **XML** | eXtensible Markup Language, 확장 가능한 마크업 언어 |
| **ClickOnce** | Microsoft의 웹 기반 애플리케이션 배포 기술 |
| **COM** | Component Object Model, Microsoft의 컴포넌트 기술 |

### 데이터베이스

| 용어 | 설명 |
|------|------|
| **RDBMS** | Relational Database Management System, 관계형 데이터베이스 관리 시스템 |
| **SQL** | Structured Query Language, 구조화된 질의 언어 |
| **PL/SQL** | Procedural Language/SQL, Oracle의 절차적 SQL 확장 |
| **Stored Procedure** | 저장 프로시저, 데이터베이스에 저장된 SQL 프로그램 |
| **Trigger** | 트리거, 특정 이벤트 발생 시 자동 실행되는 프로그램 |
| **Index** | 인덱스, 데이터 검색 성능을 향상시키는 데이터 구조 |
| **Transaction** | 트랜잭션, 하나의 논리적 작업 단위 |
| **ACID** | Atomicity, Consistency, Isolation, Durability - 트랜잭션 특성 |
| **Normalization** | 정규화, 데이터베이스 구조 최적화 과정 |
| **Sharding** | 샤딩, 대용량 데이터베이스 분산 저장 기법 |

### 보안

| 용어 | 설명 |
|------|------|
| **SSL/TLS** | Secure Sockets Layer/Transport Layer Security, 전송 계층 보안 프로토콜 |
| **HTTPS** | HyperText Transfer Protocol Secure, 보안 HTTP |
| **VPN** | Virtual Private Network, 가상 사설망 |
| **Firewall** | 방화벽, 네트워크 보안 시스템 |
| **Authentication** | 인증, 사용자 신원 확인 |
| **Authorization** | 인가, 접근 권한 부여 |
| **Encryption** | 암호화, 데이터를 안전하게 변환 |
| **Triple DES** | Triple Data Encryption Standard, 대칭 키 암호화 알고리즘 |
| **AES** | Advanced Encryption Standard, 고급 암호화 표준 |
| **Hash** | 해시, 데이터를 고정된 길이의 값으로 변환 |

## 업무 용어 (Business Terms)

### 생산 관리

| 용어 | 영문 | 설명 |
|------|------|------|
| **작업지시** | Work Order | 생산을 지시하는 문서 또는 지시사항 |
| **생산계획** | Production Plan | 생산 목표 및 일정 계획 |
| **작업실적** | Production Result | 실제 생산된 결과 기록 |
| **공정** | Process | 제품을 생산하는 단계 또는 과정 |
| **라우팅** | Routing | 제품 생산에 필요한 공정 순서 |
| **라인** | Production Line | 생산 라인, 제품을 생산하는 설비의 배열 |
| **호기** | Unit/Equipment | 개별 설비 또는 기계 |
| **투입** | Input | 생산에 필요한 자재를 투입하는 행위 |
| **산출** | Output | 생산된 결과물 |
| **UPH** | Units Per Hour | 시간당 생산량 |
| **OEE** | Overall Equipment Effectiveness | 설비종합효율성 |
| **TAT** | Turn Around Time | 처리 소요 시간 |
| **Cycle Time** | 사이클 타임 | 한 제품을 생산하는 데 소요되는 시간 |
| **Lead Time** | 리드 타임 | 주문부터 납품까지 소요되는 시간 |
| **Yield** | 수율 | 양품률, 불량률의 반대 개념 |

### 품질 관리

| 용어 | 영문 | 설명 |
|------|------|------|
| **IQC** | Incoming Quality Control | 수입검사, 입고 자재 검사 |
| **IPQC** | In-Process Quality Control | 공정검사, 생산 중 검사 |
| **OQC** | Outgoing Quality Control | 출하검사, 납품 전 검사 |
| **QC** | Quality Control | 품질관리 |
| **QA** | Quality Assurance | 품질보증 |
| **QM** | Quality Management | 품질경영 |
| **불량** | Defect | 품질 기준에 맞지 않는 제품 |
| **양품** | Good Product | 품질 기준에 맞는 제품 |
| **검사** | Inspection | 품질을 확인하는 활동 |
| **시정조치** | Corrective Action | 문제 발생 시 원인을 제거하는 조치 |
| **예방조치** | Preventive Action | 문제 발생을 예방하는 조치 |
| **AQL** | Acceptable Quality Level | 허용품질수준 |
| **SPC** | Statistical Process Control | 통계적 공정관리 |
| **6 Sigma** | 6 시그마 | 품질 관리 방법론 |

### 자재 관리

| 용어 | 영문 | 설명 |
|------|------|------|
| **자재** | Material | 생산에 사용되는 원재료, 부품 등 |
| **재고** | Inventory | 보유하고 있는 자재 또는 제품 |
| **LOT** | Lot | 동일 조건으로 생산된 제품의 묶음 |
| **시리얼** | Serial Number | 개별 제품에 부여된 고유 번호 |
| **바코드** | Barcode | 제품 식별을 위한 코드 |
| **QR코드** | QR Code | 2차원 바코드 |
| **RFID** | Radio Frequency Identification | 무선 주파수 식별 기술 |
| **입고** | Receipt | 자재를 창고에 받아들이는 행위 |
| **출고** | Issue | 자재를 창고에서 내보내는 행위 |
| **이동** | Transfer | 자재의 위치를 변경하는 행위 |
| **조정** | Adjustment | 재고 수량을 조정하는 행위 |
| **실사** | Stock Taking | 실제 재고를 파악하는 행위 |
| **안전재고** | Safety Stock | 품절 방지를 위한 최소 재고 수준 |
| **재고회전율** | Inventory Turnover | 재고가 얼마나 빨리 소진되는지 나타내는 지표 |
| **FIFO** | First In First Out | 선입선출, 먼저 입고된 것을 먼저 출고 |
| **LIFO** | Last In First Out | 후입선출, 나중에 입고된 것을 먼저 출고 |

## 데이터베이스 용어 (Database Terms)

### 테이블/엔티티

| 용어 | 설명 | 예시 |
|------|------|------|
| **마스터 데이터** | 기준정보, 시스템 운영의 기본이 되는 데이터 | 품목마스터, 거래처마스터 |
| **트랜잭션 데이터** | 업무 처리 결과 생성되는 데이터 | 작업실적, 입출고이력 |
| **이력 데이터** | 변경 이력을 관리하는 데이터 | 수정이력, 접속이력 |
| **참조 데이터** | 코드값 등 참조용 데이터 | 공통코드, 용어정보 |
| **PK** | Primary Key, 기본키 - 레코드의 고유 식별자 | WONO, ITEMCODE |
| **FK** | Foreign Key, 외래키 - 다른 테이블과의 관계 | BOM의 ITEMCODE |
| **UK** | Unique Key, 고유키 - 중복 불가한 값 | SERIAL_NO |

### 저장 프로시저 명명

| 패턴 | 설명 | 예시 |
|------|------|------|
| `PKG<모듈>_<엔티티>` | 패키지명 | PKGPRD_PROD, PKGMAT_STOCK |
| `GET_<대상>` | 조회 | GET_WORKORDER_LIST |
| `SET_<대상>` | 저장 | SET_WORKORDER |
| `DEL_<대상>` | 삭제 | DEL_WORKORDER |
| `PUT_<대상>` | 처리 | PUT_WORKORDER_CONFIRM |
