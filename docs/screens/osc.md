# 모니터링 (OSC) 모듈

## 개요

| 항목 | 내용 |
|:---|:---|
| **모듈코드** | OSC |
| **구현 화면** | 5개 |
| **주요 역할** | 대형화면 모니터링, 자재요청, 외주현황 |
| **주요 패키지** | PKGPRD_MNT, PKGPRD_PROD, PKGPRD_REPORT |

OSC 모듈은 공장 내 대형 모니터에 표시되는 실시간 모니터링 화면을 제공합니다.

## 구현 화면 목록

| 화면ID | 화면명 | 유형 | 설명 | 호출 프로시저 | 상태 |
|:---|:---|:---:|:---|:---|:---:|
| OSCA201 | 생산모니터링(대) | 모니터링 | 대형모니터용 생산현황 | PKGPRD_CURRENT.* | ✅ |
| OSCA202 | 라인모니터링 | 모니터링 | 라인별 현황 | PKGPRD_MNT.* | ✅ |
| OSCA203 | 설비모니터링 | 모니터링 | 설비 가동현황 | PKGPRD_MNT.* | ✅ |
| OSCA204 | 자재요청모니터링 | 모니터링 | 라인 자재요청 현황 | PKGPRD_PROD.GET_OSC_PRODMATERIALREQUEST | ✅ |
| OSCA205 | 외주현황 | 조회 | 외주 가공 현황 | PKGPRD_REPORT.GET_OUTSOURCING_INOUT | ✅ |

## 주요 화면 상세

### OSCA201 - 생산모니터링(대형)

공장 내 대형 디스플레이에 표시되는 종합 생산현황입니다.

| 항목 | 내용 |
|:---|:---|
| **표시정보** | 라인별 생산계획/실적/달성률, 실시간 가동현황 |
| **갱신주기** | 자동 갱신 (5분 간격) |
| **용도** | 공장 현장 대형 TV/모니터 |

### OSCA204 - 자재요청모니터링

생산라인에서 발생하는 자재 요청을 실시간으로 모니터링합니다.

| 항목 | 내용 |
|:---|:---|
| **호출 프로시저** | PKGPRD_PROD.PUT_OSC_PRODMATERIALREQUEST, GET_OSC_PRODMATERIALREQUEST |
| **주요기능** | 라인별 자재요청 등록/조회, 재고현황 확인 |

### OSCA205 - 외주현황

| 항목 | 내용 |
|:---|:---|
| **호출 프로시저** | PKGPRD_REPORT.GET_OUTSOURCING_INOUT |
| **주요기능** | 외주업체별 입출고 현황 조회 |

## 미구현 화면 (계획)

!!! warning "미구현(계획)"
    아래 기능들은 향후 개발 예정입니다. 상세 내용은 [미구현 기능](./planned-features.md)을 참조하세요.

| 기능 | 설명 |
|:---|:---|
| 품질 대시보드 | 실시간 불량률/품질 지표 대형화면 |
| 재고 대시보드 | 창고별/품목별 재고 수준 시각화 |
| 설비 가동 대시보드 | OEE/가동률 실시간 대시보드 |
| 종합 KPI 대시보드 | 생산/품질/납기/재고 종합 KPI |

## 연계 모듈

- **PRD**: 생산실적/작업지시 데이터 실시간 표시
- **MAT**: 자재 요청/재고 현황 표시
- **MNT**: 생산/설비 모니터링 데이터 공유
- **MST**: 라인/설비 마스터 참조
