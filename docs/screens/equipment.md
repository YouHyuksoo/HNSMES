## 설비기준정보

### EQM001 - 설비등록

설비의 기본 정보를 등록합니다.

#### 설비 분류

```mermaid
graph TD
    A[설비] --> B[생산설비]
    A --> C[검사설비]
    A --> D[공용설비]
    A --> E[기타설비]
    
    B --> B1[가공기]
    B --> B2[조립기]
    B --> B3[포장기]
    
    C --> C1[측정기]
    C --> C2[시험기]
```

#### 입력 항목

| 항목 | 필수 | 설명 |
|------|------|------|
| 설비번호 | Y | 고유 설비 식별번호 |
| 설비명 | Y | 설비 명칭 |
| 설비분류 | Y | 설비 분류 코드 |
| 모델명 | N | 기기 모델명 |
| 제조사 | N | 제조 회사명 |
| 제조번호 | N | 제조사 일련번호 |
| 구입일자 | N | 구매 일자 |
| 설치일자 | N | 공장 설치 일자 |
| 소속작업장 | Y | 설치된 작업장 |
| 소유구분 | Y | 자사/임대/리스 |
| 구입가격 | N | 구매 금액 |
| 사용여부 | Y | 사용/미사용 |

---

### EQM003 - 설비이력카드

설비의 전 생애주기 이력을 관리합니다.

#### 이력 구성

```mermaid
timeline
    title 설비 이력 타임라인
    section 도입기
        구입 : 구매계약
             : 입고검수
        설치 : 설치완료
             : 시운전
    section 운영기
        정상가동 : 생산투입
        점검 : 정기점검
        수리 : 고장수리
    section 폐기기
        퇴역 : 생산불가
        폐기 : 매각/폐기
```

    진행 --> 완료: 수리완료
    진행 --> 외주: 외주의뢰

### EQM021 - 설비가동현황

설비 가동률을 분석합니다.

#### 가동률 계산

```
가동률(%) = (가동시간 / 총시간) × 100
성능가동률 = (실제생산량 / 이론생산량) × 100
종합효율(OEE) = 가동률 × 성능가동률 × 양품률
```

#### 가동현황 차트

```mermaid
xychart-beta
    title "작업장별 설비 가동률"
    x-axis ["WC01", "WC02", "WC03", "WC04", "WC05"]
    y-axis "가동률(%)" 0 --> 100
    bar [85, 92, 78, 88, 95]
    line [85, 85, 85, 85, 85]
```
