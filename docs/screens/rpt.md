# 리포트 (RPT) 모듈

## 개요

| 항목 | 내용 |
|:---|:---|
| **모듈코드** | RPT |
| **구현 화면** | 22개 |
| **주요 역할** | 생산일보/월보, 불량현황, 설비가동, KPI 분석 |
| **주요 패키지** | PKGHNS_REPORT, PKGPRD_REPORT |
