## 검사관리

### QCM001 - 수입검사등록

원자재 입고 시 품질 검사 결과를 등록합니다.

#### 검사 유형

```mermaid
graph TD
    A[품질검사] --> B[수입검사]
    A --> C[공정검사]
    A --> D[최종검사]
    A --> E[출하검사]
    
    B --> B1[원자재검사]
    B --> B2[부자재검사]
    C --> C1[공정중검사]
    C --> C2[공정완료검사]
```

#### 입력 항목

| 항목 | 필수 | 설명 |
|------|------|------|
| 검사일자 | Y | 검사 실시일 |
| 입고번호 | Y | 연관 입고 번호 |
| 품목코드 | Y | 검사 대상 품목 |
| LOT번호 | N | 검사 대상 LOT |
| 검사수량 | Y | 총 검사 대상 수량 |
| 샘플수량 | Y | 실제 검사 수량 |
| 합격수량 | Y | 합격 판정 수량 |
| 불량수량 | Y | 불합격 수량 |
| 검사결과 | Y | 합격/불합격/조건부 |
| 검사자 | Y | 검사 담당자 |

#### 검사 결과 처리

```mermaid
stateDiagram-v2
    [*] --> 대기: 입고
    대기 --> 검사중: 검사시작
    검사중 --> 합격: 검사완료
    검사중 --> 불합격: 불량발생
    합격 --> 입고완료: 입고처리
    불합격 --> 반품: 반품처리
    불합격 --> 특采: 특별채택
```

## SPC 관리도

### QCM019 - X-Bar 관리도

공정의 중심 경향을 관리합니다.

#### 관리도 구성

```mermaid
graph TD
    A[X-Bar 관리도] --> B[중심선 CL]
    A --> C[상한관리한계 UCL]
    A --> D[하한관리한계 LCL]
    
    B --> B1[평균의 평균]
    C --> C1[Xbar + A2Rbar]
    D --> D1[Xbar - A2Rbar]
```

#### 판정 규칙 (Zone 규칙)

| 규칙 | 설명 |
|------|------|
| 1 | 1점이 관리한계 외부 |
| 2 | 연속 9점이 중심선 동측 |
| 3 | 연속 6점이 지속적으로 상승/하강 |
| 4 | 연속 14점이 상하 교대 |

### QCM022 - 공정능력분석

공정의 품질 능력을 평가합니다.

#### 능력지수

| 지수 | 설명 | 판정기준 |
|------|------|----------|
| Cp | 공정능력지수 | Cp ≥ 1.33 양호 |
| Cpk | 공정능력지수(고려) | Cpk ≥ 1.33 양호 |
| Pp | 장기공정능력 | Pp ≥ 1.33 양호 |
| Ppk | 장기공정능력(고려) | Ppk ≥ 1.33 양호 |

#### 능력지수 계산

```
Cp = (USL - LSL) / 6σ
Cpk = min[(USL - μ)/3σ, (μ - LSL)/3σ]
```
