    변경 이력 (Changelog)¶

    HNSMES 시스템의 버전별 변경 이력입니다.

    버전 규칙¶

    버전 번호는 MAJOR.MINOR.PATCH 형식을 따릅니다.

    위치 설명 예시
    MAJOR 주요 기능 변경, 호환성 없는 변경 1.0.0 → 2.0.0
    MINOR 기능 추가, 하위 호환성 유지 1.0.0 → 1.1.0
    PATCH 버그 수정, 하위 호환성 유지 1.0.0 → 1.0.1

    [2.1.0] - 2024-12-15¶

    추가¶

    • 생산계획 시뮬레이션 기능 (PLAN015)
    • AI 기반 불량예측 대시보드 (MST026)
    • 모바일 생산실적 입력 앱 연동
    • 다중 언어 지원 (영어, 중국어)

    개선¶

    • 대시보드 로딩 속도 40% 향상
    • 리포트 출력 속도 개선
    • 재고 조회 성능 최적화

    수정¶

    • 설비비가동 등록 시 간헐적 오류 수정
    • 수입검사 LOT 중복 체크 로직 수정
    • 작업지시 확정 시 권한 체크 오류 수정

    보안¶

    • 비밀번호 정책 강화 (12자리 이상)
    • 2차 인증(OTP) 지원 추가
    • 세션 관리 로직 개선

    [2.0.5] - 2024-10-30¶

    수정¶

    • MSSQL 데이터베이스 연결 오류 수정
    • 대시보드 실시간 갱신 오류 수정
    • 일부 화면에서 날짜 포맷 오류 수정

    개선¶

    • 로그인 속도 개선
    • 대용량 데이터 조회 성능 개선

    [2.0.0] - 2024-08-01¶

    주요 변경사항¶

    • 데이터베이스 Oracle → PostgreSQL 마이그레이션
    • UI 프레임워크 전면 개편 (WinForms → WPF)
    • REST API 기반 아키텍처 도입
    • 클라우드 지원 추가

    추가¶

    • 실시간 알림 시스템
    • Web 기반 모니터링 대시보드
    • 외주생산 관리 모듈
    • E-signature 기능

    제거¶

    • 레거시 리포트 뷰어 지원 종료
    • Windows XP 지원 종료

    [1.5.3] - 2024-05-20¶

    수정¶

    • BOM 정전개 조회 시 성능 저하 수정
    • 작업지시 복사 시 날짜 오류 수정
    • 재고 이동 후 잔량 계산 오류 수정

    개선¶

    • 그리드 필터링 기능 강화
    • 엑셀 내보내기 속도 개선

    [1.5.0] - 2024-03-15¶

    추가¶

    • 설비예측보전 모듈 (EQM023-EQM025)
    • 품질관리 SPC 모듈 고도화
    • 외주가공비 관리 기능

    개선¶

    • MRP 실행 속도 50% 향상
    • 설비가동현황 UI 개선
    • 다중 공장 지원 강화

    예정된 변경사항¶

    [2.2.0] - 2025년 3월 예정¶

    • AI 기반 수요예측 모듈
    • 디지털 트윈 연동
    • 블록체인 기반 이력관리 (PoC)

    [3.0.0] - 2025년 하반기 예정¶

    • 클라우드 네이티브 아키텍처 전환
    • 마이크로서비스 아키텍처 도입
    • Web 기반 UI 전면 전환

    업그레이드 가이드¶

    1.x → 2.0 업그레이드¶

    1. 데이터베이스 백업
2. 기존 프로그램 제거
3. 신규 버전 설치
4. 데이터 마이그레이션 실행
5. 설정 파일 복원
6. 기능 테스트

    주의사항¶

    • 업그레이드 전 반드시 전체 백업 수행
    • 업그레이드는 계획된 점검 시간에 진행
    • 롤백 계획 사전 수립

    2.0.x Web 대시보드 IE 미지원 Edge/Chrome 사용 -
    1.5.x Windows 11 일부 UI 깨짐 호환성 모드 사용 2.0
